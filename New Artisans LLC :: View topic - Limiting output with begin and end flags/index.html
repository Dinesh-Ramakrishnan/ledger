Bogart/
I've found some difficulty in using the -b and -e flags too.. You might try specifying dates in exactly the same format as your ledger entries to see if that helps.
Because of past problems (and the fact I prefer dd/mm/yyyy as a date format) I usually use the more long-winded syntax

ledger -l 'd<=[2007/01/01]' reg

Which seems more reliable at parsing dates.

There used to be command-line options to tell ledger what your date format is- not sure if they still exist or not.
